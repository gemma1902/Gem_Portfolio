Django Scout Inventory App

A full-stack Django web app designed to help scout groups efficiently manage their equipment and supplies. The system allows users to add, edit and track items such as camping gear, uniforms and tools, ensuring accurate stock control and accountability. Built with Python (Django), HTML, CSS and SQLite, it provides a simple, intuitive interface for everyday use.

Vancouver Business Opportunity Finder

An interactive web application that helps identify promising locations for new restaurants and food businesses across Vancouver using City of Vancouver open data. The dashboard combines competitor density, area activity, property values and development data into a custom opportunity score, while allowing users to explore neighbourhoods and adjust the scoring themselves. Built with Python, pandas, Streamlit and Plotly, the project involved data cleaning, feature engineering, data integration and interactive dashboard development.

BC Care Capacity

A regional data analysis exploring gaps in long-term care capacity across British Columbia. Using public data from BC's Ministry of Health and PHSA, this project compares the projected senior population with the number of available long-term care beds across BC's five health authorities to identify which regions are best and least prepared to support their aging populations.
